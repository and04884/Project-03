# Project Retrospective  
## Explore Waco, Texas – Tourism Website

This retrospective reflects on the development of the *Explore Waco, Texas* tourism website. Following PMUPM Chapter 7, it answers the three key questions.

---

## 1. What Went Well

- The project scope was clear and prevented scope creep.  
- HTML structure and CSS organization made development smooth.  
- Responsive design worked effectively across devices.  
- Navigation between pages was intuitive and functional.  
- Documentation remained organized and was completed on time.

---

## 2. What Went Wrong

- Finding high‑quality images of Waco took longer than expected.  
- Some CSS elements required multiple revisions for mobile layouts.  
- The Google Maps embed initially caused spacing issues.  
- Styling took more time than anticipated, pushing testing later.

---

## 3. What I Would Do Differently

- Begin styling earlier to avoid last‑minute adjustments.  
- Create a content inventory before coding.  
- Prototype page layouts before writing content.  
- Test GitHub Pages deployment earlier.  
- Use a more modular CSS approach for easier updates.
