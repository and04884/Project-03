# Project Plan  
## Explore Waco, Texas – Tourism Website

This project plan outlines the work required to build the *Explore Waco, Texas* tourism website. It includes a TAME risk analysis and a Work Breakdown Structure (WBS) based on PMUPM Chapter 4.

---

## TAME Risk Analysis

### 1. Layout Breaks on Different Screen Sizes  
- **Type:** Technical  
- **Impact:** Medium  
- **Likelihood:** Medium  
- **Strategy:** Mitigate  
- **Actions:**  
  - Use CSS media queries  
  - Test on desktop, tablet, and mobile  
  - Validate layout in multiple browsers  

### 2. Slow Loading Due to Images or Embedded Map  
- **Type:** Performance  
- **Impact:** Medium  
- **Likelihood:** Medium  
- **Strategy:** Mitigate  
- **Actions:**  
  - Compress images  
  - Use efficient file formats  
  - Test Google Maps embed performance  

### 3. Limited Time to Complete All Deliverables  
- **Type:** Schedule  
- **Impact:** High  
- **Likelihood:** Medium  
- **Strategy:** Mitigate  
- **Actions:**  
  - Follow WBS task order  
  - Complete core pages first  
  - Leave styling refinements for last  

### 4. Difficulty Finding High‑Quality Images  
- **Type:** Resource  
- **Impact:** Low  
- **Likelihood:** Medium  
- **Strategy:** Accept  
- **Actions:**  
  - Use free image sources  
  - Replace images later if better ones are found  

### 5. GitHub Pages Deployment Issues  
- **Type:** Technical  
- **Impact:** Medium  
- **Likelihood:** Low  
- **Strategy:** Tolerate  
- **Actions:**  
  - Follow GitHub Pages documentation  
  - Test deployment early  
  - Validate links after publishing
